Detailed Description

this class hosts all the specific adapters and manages them. It provides APIs for the users to initialize, access and interact with the adapters that they are interested in.

Each (potentially) useful adapter needs to be registered here with the macro REGISTER_ADAPTER.
The AdapterManager is a singleton.
